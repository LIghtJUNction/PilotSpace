# PilotSpace
AI原生的工作区革命

# 目前还在开发中

## 预配置
- 本项目计划对大量常见软件进行预配置
yolk.rhai可以充分自定义你喜欢的组合
默认shell为zsh
使用hyprland
使用quickshell
终端为Kitty

## 安装
本项目使用yolk管理配置文件
安装可分为以下几个步骤
- 安装所有依赖
- 使用yolk绑定到pilotspace-eggs

- yolk pull
- yolk sync

## 自定义
如何保留自定义配置呢？
我的建议是：
- 直接fork pilot-space
按需修改

## 为什么选择yolk？
yolk的作者是eww的作者，我浏览eww仓库时发现作者做了这个软件，我体验了一下，确实挺好用的。
这里还有一些[替代品］(https://dotfiles.github.io/tutorials/)
- yolk
yolk支持模板等高级功能
更新更加方便

## 不同组合
安装配置文件就和孵蛋一样
- 更新yolk.rhai文件即可轻松禁用！
